Color Theme
Background Color:

Dark Slate: #1A1A2E (a rich dark blue-black)
Primary Text Color:

Soft White: #EAEAEA (for most text)
Secondary Text Color:

Light Gray: #B0B0B0 (for less important text)
Accent Color:

Bright Aqua: #00BFFF (for buttons and highlights)
Hover Color:

Deeper Aqua: #009ACD (for hover states)
Border Color:

Semi-Transparent Gray: rgba(255, 255, 255, 0.1) (for subtle borders)
Shadow Color:

Soft Black: rgba(0, 0, 0, 0.5) (for box shadows)
Example Usage
Background: Use #1A1A2E for your body and section backgrounds.
Text: Use #EAEAEA for main text, and #B0B0B0 for secondary text.
Buttons: Use #00BFFF for button backgrounds, and change to #009ACD on hover.
Borders: Apply rgba(255, 255, 255, 0.1) for borders around elements.
Shadows: Use rgba(0, 0, 0, 0.5) for box shadows to add depth.
